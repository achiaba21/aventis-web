Compte Proprietaire

Systeme de gestion de compte pour les proprietaires avec soldes, historique et retraits

Resume

Cette feature permet aux proprietaires de :

  • Consulter leur solde - Affichage du solde disponible, en attente et verrouille
  • Voir l'historique - Liste des transactions avec filtrage par type (credit/debit)
  • Demander un retrait - Formulaire de demande avec validation du montant

Architecture

Modeles

CompteProprietaire

Compte principal du proprietaire etendant la classe abstraite Compte. Contient le solde disponible et reference vers CompteAttente.

CompteAttente

Gere les fonds en attente (solde) et verrouilles (avances de reservation).

Transaction

Represente une transaction (CREDIT ou DEBIT) avec montant, date et statut.

DemandeRetrait

Demande de retrait avec suivi du statut (EN_ATTENTE, VALIDEE, REFUSEE).

Flux de donnees

UI (Screen) → CompteBloc → CompteRepository → CompteApiService → API Server

API Endpoints

Methode Endpoint Description
GET /api/v1/comptes/proprietaire Recuperer le compte du proprietaire connecte
GET /api/v1/comptes/transactions Liste des transactions avec pagination
POST /api/v1/comptes/retraits Creer une demande de retrait
GET /api/v1/comptes/retraits Historique des demandes de retrait

Composants UI

CompteScreen

Ecran principal affichant :

  • Banniere d'alerte si compte suspendu
  • SoldeCard - Carte principale avec solde et bouton retrait
  • MiniMetricCard (x2) - Montants en attente et verrouille
  • Liste des transactions recentes (5 dernieres)
  • Lien vers l'historique complet

HistoriqueScreen

Ecran d'historique avec :

  • Filtres par type (Tous / Credits / Debits)
  • Regroupement par mois
  • Pull-to-refresh

RetraitForm

Bottom sheet modal avec :

  • Affichage du solde disponible
  • Champ de saisie du montant
  • Bouton "Tout retirer"
  • Validation (montant > 0 et ≤ solde)

Utilisation

Acces

Depuis le profil proprietaire : Mon Compte (icone compte bancaire)

Exemple d'integration

// Charger le compte
context.read<CompteBloc>().add(LoadCompte());

// Demander un retrait
context.read<CompteBloc>().add(DemanderRetrait(montant: 50000));

// Ecouter les etats
BlocBuilder<CompteBloc, CompteState>(
  builder: (context, state) {
    if (state is CompteLoaded) {
      return Text('Solde: ${state.compte.solde} FCFA');
    }
    return CircularProgressIndicator();
  },
)
