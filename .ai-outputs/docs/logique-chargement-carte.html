Logique Chargement Carte

Affichage des residences sur la carte depuis le cache ResidenceBloc

Livree v1.0

Objectif

Modifier l'ecran carte pour afficher les residences provenant du cache ResidenceBloc, en filtrant celles sans coordonnees GPS. L'utilisateur peut centrer la carte sur sa position GPS ou choisir un point manuellement.

Fonctionnalites

  • Source unique : Les residences proviennent du cache ResidenceBloc (pas d'appel API separe)
  • Filtrage GPS : Seules les residences avec address.hasExactLocation == true sont affichees
  • Conversion : Factory MapResidence.fromResidence() pour transformer les residences
  • Geolocalisation : Centrage sur la position utilisateur
  • Selection manuelle : Long press sur la carte pour choisir un point

Architecture

ResidenceBloc ─────> ResidenceRepository ─────> API /residences (cache) │ └──> Liste Residence (avec appartements) │ ▼ MapExploreScreen │ ├──> Filtre celles avec coordonnees GPS │ (r.address?.hasExactLocation == true) │ └──> Convertit en MapResidence pour affichage (MapResidence.fromResidence(r))

Factory MapResidence.fromResidence()

factory MapResidence.fromResidence(Residence residence) {
  final address = residence.address;
  final appartements = residence.appartements ?? [];

  final prices = appartements
      .where((a) => a.prix != null)
      .map((a) => a.prix!)
      .toList();

  final minP = prices.isNotEmpty
      ? prices.reduce((a, b) => a < b ? a : b)
      : null;

  return MapResidence(
    id: residence.id,
    nom: residence.nom,
    displayLat: address?.lat,
    displayLongi: address?.longi,
    appartementCount: appartements.length,
    minPrice: minP,
    // ...
  );
}

Chargement depuis ResidenceBloc

void _loadResidencesFromBloc() {
  final resBloc = context.read<ResidenceBloc>();
  final state = resBloc.state;

  if (state is ResidencesLoaded) {
    _updateMapResidences(state.residences);
  } else {
    resBloc.add(LoadResidences());
  }
}

void _updateMapResidences(List<Residence> residences) {
  setState(() {
    _mapResidences = residences
        .where((r) => r.address?.hasExactLocation == true)
        .map((r) => MapResidence.fromResidence(r))
        .toList();
  });
}
