Description

Refonte totale de l'identite visuelle de l'app Flutter Asfar (multi-role locataire / proprietaire / demarcheur), basee sur un prototype HTML React livre par le design produit. Le theme clair historique disparait au profit d'un theme dark premium avec accent or chaud (#E8B86B).

Stack & contraintes

  • Stack : Flutter 3.7+ / Dart 3, BLoC, Hive, Spring Boot local
  • Plateformes : iOS 13+ / Android 10+
  • Localisation : Cote d'Ivoire / FCFA / fr-FR
  • Perimetre : UI uniquement (BLoCs, models, repos, services, Hive, backend = inchanges)

Strategie : "Swap des entrailles"

Plutot que de creer un namespace parallele AsfarColors / AsfarButton coexistant avec l'ancien, la refonte preserve les noms publics et signatures et remplace uniquement les valeurs et l'implementation interne.

Resultat : les 2 352 references existantes (1 655 sur AppColors + 697 sur TextSeed) a travers les 121 ecrans basculent automatiquement au moment du commit du fichier app_colors.dart. Aucun callsite a modifier au cas par cas.
