Geocodage Address

Systeme de coordonnees geocodees pour la protection de la vie privee

Valide v1.0 2025-12-27

Resume

Cette feature ajoute un systeme de coordonnees geocodees au modele Address pour permettre l'affichage des residences sur la carte tout en protegeant la vie privee des proprietaires.

Principe

  • Coordonnees exactes (lat/longi) : Position reelle de la residence
  • Coordonnees geocodees (geoLat/geoLongi) : Position approximative (centre du quartier/commune)
  • Le serveur decide quelles coordonnees envoyer selon le contexte

Regles Metier

Contexte Coordonnees envoyees
Visiteur / Locataire sans reservation Geocodees (approximatives)
Locataire avec reservation PAYEE Exactes
Proprietaire (sa propre residence) Exactes

Architecture

SERVEUR +-----------------------------------------+ | Address | | +-- lat / longi (exactes) | | +-- geoLat / geoLongi (geocodees) | | | | Logique d'envoi : | | Si reservation PAYEE OU proprietaire | | -> envoyer lat/longi | | Sinon | | -> envoyer geoLat/geoLongi | +-----------------------------------------+ | v CLIENT (Flutter) +-----------------------------------------+ | Recoit : displayLat / displayLongi | | (ne sait pas si exact ou geocode) | | | | Si geoLat/geoLongi vides : | | -> GeocodingService.geocodeAddress | | -> AddressService.updateGeocoded | +-----------------------------------------+

API Reference

Address (Modele)

Nouveaux champs

double? geoLat;      // Latitude geocodee
double? geoLongi;    // Longitude geocodee

Nouveaux getters

bool get hasGeocodedLocation;   // true si geoLat et geoLongi sont definis
LatLng? get geocodedLocation;   // Retourne LatLng geocode ou null
LatLng? get displayLocation;    // Retourne exactes ?? geocodees

GeocodingService

geocode(String query)

Geocode une requete textuelle via l'API Nominatim.

Future<GeocodingResult?> geocode(String query)

// Exemple
final result = await GeocodingService.instance.geocode('Cocody, Abidjan');
print(result?.latLng); // LatLng(5.34, -4.02)

geocodeAddress(Address address)

Geocode une Address a partir de son nom + commune.

Future<LatLng?> geocodeAddress(Address address)

// Exemple
final coords = await GeocodingService.instance.geocodeAddress(residence.address);

reverseGeocode(LatLng coords)

Geocodage inverse : coordonnees vers adresse textuelle.

Future<String?> reverseGeocode(LatLng coords)

// Exemple
final address = await GeocodingService.instance.reverseGeocode(LatLng(5.34, -4.02));
print(address); // "Cocody, Abidjan, Cote d'Ivoire"

autocomplete(String query, {int limit})

Recherche avec autocompletion (plusieurs resultats).

Future<List<GeocodingResult>> autocomplete(String query, {int limit = 5})

// Exemple
final results = await GeocodingService.instance.autocomplete('Coco');
// [GeocodingResult(Cocody), GeocodingResult(Coconut Grove), ...]

AddressService

geocodeAndSave(Address address)

Geocode une adresse et sauvegarde le resultat sur le serveur.

Future<Address> geocodeAndSave(Address address)

// Exemple
final updatedAddress = await AddressService.instance.geocodeAndSave(address);
// address.geoLat et address.geoLongi sont maintenant definis
// Le serveur a ete mis a jour via PATCH

updateGeocodedCoordinates(int id, double lat, double lon)

Met a jour les coordonnees geocodees sur le serveur.

Future<bool> updateGeocodedCoordinates(int addressId, double geoLat, double geoLongi)

// Appelle: PATCH /addresses/{id}/geocode
// Body: { "geoLat": 5.34, "geoLongi": -4.02 }

Exemples d'utilisation

Affichage sur la carte

// Dans le widget de carte
final address = residence.address;

// Utiliser displayLocation qui priorise exact > geocode
final position = address?.displayLocation;

if (position != null) {
  // Afficher le marqueur
  Marker(
    point: position,
    child: Icon(Icons.location_on),
  )
}

Geocodage a la volee si manquant

// Si les coords geocodees sont manquantes
if (!address.hasGeocodedLocation) {
  // Geocoder et sauvegarder
  await AddressService.instance.geocodeAndSave(address);
}

Recherche avec autocompletion

// Dans un champ de recherche
TextField(
  onChanged: (query) async {
    if (query.length >= 3) {
      final suggestions = await GeocodingService.instance.autocomplete(query);
      // Afficher les suggestions
    }
  },
)

Endpoint Serveur Requis

PATCH /addresses/{id}/geocode

Met a jour les coordonnees geocodees d'une adresse.

// Request
PATCH /addresses/123/geocode
Content-Type: application/json

{
  "geoLat": 5.3364,
  "geoLongi": -4.0267
}

// Response
200 OK
