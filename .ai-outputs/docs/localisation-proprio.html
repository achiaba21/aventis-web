Localisation Proprio

Differentiation proprio/locataire et edition de localisation

Implemente v1.0 2025-12-27

Resume

Cette feature differencie l'affichage de la section carte selon le role (proprietaire vs locataire) et permet au proprietaire d'ajouter/modifier la localisation de sa residence via 3 methodes.

Objectifs

  • Le proprio ne voit plus "Reserver" sur ses propres residences
  • Le proprio peut ajouter une localisation via 3 methodes
  • Le proprio peut modifier une localisation existante
  • Le locataire voit "Localisation approximative" si pas de coords exactes

Etats de ResidenceMapSection

Etat 1 : Proprio AVEC coordonnees

Carte avec marker + FAB "Modifier" en bas a droite

ResidenceMapSection(
  residence: residence,
  isOwner: true,
  onEditLocation: _handleEditLocation,
)

Etat 2 : Proprio SANS coordonnees

Placeholder "Localisation non renseignee" avec bouton "Ajouter"

Etat 3 : Locataire (carte approximative)

Carte centree sur commune + message "Localisation approximative"

Plus de bouton "Reserver maintenant"

Etat 4 : Aucune localisation

Placeholder standard avec action reservation

LocationEditBottomSheet

Bottom sheet modal avec 3 onglets pour la saisie de localisation :

Onglet Recherche

  • Champ de recherche avec autocompletion
  • Utilise GeocodingService.autocomplete()
  • Bouton "Utiliser ma position actuelle"
  • Liste des suggestions cliquables

Onglet Carte

  • Carte interactive (tap pour selectionner)
  • Marker affiche a la position selectionnee
  • Bouton geolocalisation

Onglet Manuel

  • Champs Latitude et Longitude
  • Validation des plages (-90/90, -180/180)
  • Bouton position actuelle

Usage

final result = await LocationEditBottomSheet.show(
  context,
  initialAddress: residence.address,
);

if (result != null) {
  // result.latitude, result.longitude, result.displayName
}

Regles Metier

Regle Description
RM1 Le proprio voit TOUJOURS les coordonnees exactes de sa residence
RM2 Le proprio peut ajouter/modifier la localisation a tout moment
RM3 Le locataire sans reservation payee voit la carte centree sur la commune
RM4 Le locataire avec reservation payee voit les coordonnees exactes
RM5 La saisie de localisation propose 3 methodes
