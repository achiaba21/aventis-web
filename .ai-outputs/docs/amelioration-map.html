Feature: Amelioration Map

Modernisation de l'affichage des cartes avec design epure et nouvelles fonctionnalites

Valide Score: 92/100

Resume

Cette feature ameliore l'experience cartographique de l'application Asfar avec :

  • Design moderne style Airbnb avec markers affichant les prix
  • Tuiles Stadia Maps en mode sombre pour un rendu epure
  • Bouton de geolocalisation avec animation
  • Recherche par zone circulaire
  • Configuration centralisee des parametres map

Composants Crees

MapConfig

Configuration centralisee

lib/config/map_config.dart
  • URLs des tuiles Stadia
  • Couleurs des markers
  • Dimensions et animations
  • Parametres de zoom

CustomMapMarker

Marker moderne avec prix

lib/widget/map/custom_map_marker.dart
  • Mode normal compact (25K)
  • Mode selectionne etendu
  • Mode cluster avec pulse
  • Animations de tap

MapStyleLayer

Tuiles stylisees

lib/widget/map/map_style_layer.dart
  • Stadia Maps integration
  • Support dark/light mode

LocationButton

FAB geolocalisation

lib/widget/map/location_button.dart
  • Etat inactif/searching/centered
  • Animation pulse

ZoneSelector

Bottom sheet zones

lib/widget/map/zone_selector.dart
  • Selection cercle/libre
  • Slider rayon (0.5-5 km)
  • Boutons recherche/effacer

Fichiers Modifies

  • lib/screen/client/locataire/map/map_explore_screen.dart Modifie
  • lib/widget/residence/residence_map_section.dart Modifie

Utilisation

CustomMapMarker

CustomMapMarker(
  price: 25000,        // Prix en FCFA
  isSelected: false,   // Etat selection
  isCluster: false,    // Mode cluster
  clusterCount: 5,     // Nombre dans cluster
  onTap: () => {},     // Callback
)

MapStyleLayer

FlutterMap(
  children: [
    const MapStyleLayer(isDarkMode: true),
    MarkerLayer(markers: [...]),
  ],
)

LocationButton

LocationButton(
  state: LocationButtonState.searching,
  onPressed: () => _centerOnUserLocation(),
)
